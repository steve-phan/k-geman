# K-Geman - German Grammar Learning App

An Android application built with Kotlin to help learners study German grammar.

## Features

### 1. Verben mit Präpositionen (Verbs with Prepositions)
Learn German verbs that require specific prepositions, including:
- The verb and its preposition
- The grammatical case (Akkusativ, Dativ, or Genitiv)
- Example sentences in German
- English translations

### 2. Artikel (Articles - der/die/das)
Practice German articles with:
- Nouns with their correct articles (der, die, das)
- Plural forms
- English translations
- Color-coded by gender (masculine, feminine, neuter)

### 3. Haupt- und Nebensätze (Main and Subordinate Clauses)
Master German sentence structure with:
- Different clause types (main, subordinate, compound)
- Common conjunctions (weil, dass, wenn, obwohl, etc.)
- Complete sentence examples
- English translations
- Explanations of word order rules

## Build Instructions

1. Clone the repository
2. Open the project in Android Studio
3. Build and run on an emulator or physical device (API 24+)

## Requirements

- Android SDK 24+
- Kotlin 1.9.0+
- Android Studio (recommended)
